# data-tip.css

## Usage

Simply write like this in your HTML:

```html
<button class="data-tip data-tip-bottom" data-tip="Tips To Show">My Custom Button</button
```

Position your tip:

```html
data-tip-top
data-tip-bottom
data-tip-left
data-tip-right
```

Colorful your tip:

```html
data-tip-success
data-tip-warning
data-tip-danger
data-tip-info
```

Anti-animation:

```html
data-tip-no-animation
```
